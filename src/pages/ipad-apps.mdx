## Coding on iPad

In short: Use a remote server, connect to it with Blink or ShellFish, use ShellFish for Files integration, Inspect for browser dev tools, Working Copy for Git, Runestone for plain text editing.

For design, Figurative + Fontcase give you a complete Figma setup, leagues more capable than using Figma in Safari.
